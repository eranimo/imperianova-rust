# ImperiaNova

Grand strategy game and historical simulator.

## Architecture
- Game engine: bevy
- Pathfinding: petgraph
- AI
    - Utility AI: big-brain
- World generation
    - Noise: noise-rs
    - Hex grid: chickenwire
